# 🚀 Portfólio de João Pedro

Portfólio profissional moderno e responsivo desenvolvido em HTML5, CSS3 e JavaScript puro. Perfeito para atrair clientes e apresentar seus projetos de desenvolvimento web.

## ✨ Características

- ✅ **Design Moderno**: Gradientes, animações e transições suaves
- ✅ **100% Responsivo**: Funciona perfeitamente em mobile, tablet e desktop
- ✅ **Performance Otimizada**: Carregamento rápido e SEO-friendly
- ✅ **Sem Dependências**: Apenas HTML, CSS e JavaScript puro
- ✅ **Fácil de Customizar**: Códigos bem organizados e comentados
- ✅ **Animações Atraentes**: Efeitos ao scroll e hover states

## 📋 Seções Incluídas

1. **Navbar Fixa**: Navegação suave entre seções
2. **Hero Section**: Apresentação impactante com Call-to-Action
3. **Sobre Mim**: Informações profissionais e habilidades
4. **Serviços**: Descrição dos serviços oferecidos (6 cards)
5. **Projetos**: Galeria de projetos destacados com tags
6. **Contato**: Múltiplas formas de contato (WhatsApp, Email, LinkedIn)
7. **Footer**: Rodapé profissional

## 🛠️ Tecnologias Utilizadas

- HTML5
- CSS3 (com variáveis CSS, Grid, Flexbox)
- JavaScript Vanilla (Intersection Observer, Event Listeners)

## 📱 Responsividade

O site é totalmente responsivo com breakpoints para:
- Desktop (1200px+)
- Tablet (768px - 1199px)
- Mobile (até 767px)

## 🚀 Como Usar

1. Clone o repositório
2. Abra `index.html` no seu navegador
3. Customize os textos, cores e informações conforme necessário

## 📝 Customizações Importantes

### Mudar Informações Pessoais
- Atualize nome e descrição no `index.html`
- Adicione seus links de WhatsApp, Email e LinkedIn

### Adicionar/Remover Projetos
- Edite a seção `#projetos` no `index.html`
- Personalize imagens e descrições

### Modificar Cores
- Altere as variáveis CSS em `styles.css` (`:root`)

### Adicionar Mais Seções
- Copie a estrutura de cards existentes
- Mantenha o padrão de classes para herdar estilos

## 📊 Otimizações Implementadas

- ⚡ Lazy loading com Intersection Observer
- 🎯 Meta tags para melhor SEO
- 📱 Mobile-first approach
- 🎨 CSS Grid e Flexbox para layouts modernos
- ⏱️ Transições suaves (0.3s)
- 🔄 Animações ao scroll

## 💡 Dicas para Atrair Mais Clientes

1. **Atualize regularmente**: Adicione novos projetos conforme completa
2. **Use imagens de qualidade**: Melhore a seção de projetos com screenshots
3. **Adicione testimoniais**: Considere adicionar uma seção com feedback de clientes
4. **Call-to-Actions claros**: Os botões de contato devem ser destacados
5. **Caso de sucesso**: Descreva resultados que seus projetos geraram
6. **Blog/Artigos**: Considere adicionar um blog com dicas de desenvolvimento
