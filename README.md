#  Portfólio de João Pedro

Portfólio profissional moderno e responsivo desenvolvido em HTML5, CSS3 e JavaScript puro.

##  Características

-  **Design Moderno**: Gradientes, animações e transições suaves
-  **100% Responsivo**: Funciona perfeitamente em mobile, tablet e desktop
-  **Performance Otimizada**: Carregamento rápido e SEO-friendly
-  **Sem Dependências**: Apenas HTML, CSS e JavaScript puro
-  **Fácil de Customizar**: Códigos bem organizados e comentados
-  **Animações Atraentes**: Efeitos ao scroll e hover states

##  Seções Incluídas

1. **Navbar Fixa**: Navegação suave entre seções
2. **Hero Section**: Apresentação impactante com Call-to-Action
3. **Sobre Mim**: Informações profissionais e habilidades
4. **Serviços**: Descrição dos serviços oferecidos (6 cards)
5. **Projetos**: Galeria de projetos destacados com tags
6. **Contato**: Múltiplas formas de contato (WhatsApp, Email, LinkedIn)
7. **Footer**: Rodapé profissional

##  Tecnologias Utilizadas

- HTML5
- CSS3 (com variáveis CSS, Grid, Flexbox)
- JavaScript Vanilla (Intersection Observer, Event Listeners)

##  Responsividade

O site é totalmente responsivo com breakpoints para:
- Desktop (1200px+)
- Tablet (768px - 1199px)
- Mobile (até 767px)

##  Paleta de Cores

- **Secundária**: #764ba2 (Roxo)
